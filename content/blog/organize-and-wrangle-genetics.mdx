## Overview

This document will focus on the planning, organization, and development of the genetics data used for VariantAlleles.com.

### ACMG Guidelines

There are three versions of the ACMG guidelines that are relevant to VariantAlleles.com

1. V3 - This corresponds to the 2015 version of the ACMG guidelines.
2. V3_5 - This corresponds to the 2015 version of the acmg guidelines with optional bayes points integration.
3. V4 - This corresponds to the unpublished 2024-2025 version of the ACMG guidelines.
